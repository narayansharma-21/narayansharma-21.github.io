Video Game Sales

Narayan Sharma

Motivation

In this assignment, I looked at the different trends of video game sales as I wanted to see how a specific console could cause a change in price each year. Studying the different trends of video game sales can help some predict the different sale costs for the future.

Data

For this assignment, I have chosen the "Video Game Sales" dataset from Kaggle (https://www.kaggle.com/gregorut/videogamesales). This dataset contains information on video game sales from 1980 through 2016, including sales numbers, publisher, platform, and region. I chose this dataset because I am interested in the video game industry and wanted to explore trends in sales over time. After exploring the dataset in Excel, I found that there were some missing values that needed to be cleaned up. I removed rows with missing values and also removed any duplicates. One interesting finding from the data is that the top-selling video game of all time is "Wii Sports" with over 82 million copies sold. Another interesting finding is that the platform with the most sales is the PlayStation 2, with over 1.2 billion copies sold.

Data Visualization(s)

First Visualization

For my first visualization, I created a stacked area chart showing the total sales by platform over time. I chose this chart type because it allows me to see how the sales of different platforms have changed over time and how they compare to each other. I used color to encode the platform, with each platform having its own color. I chose a color scheme with bright and contrasting colors to make the chart visually appealing and easy to read. The chart is interactive, allowing the user to hover over each point to see the exact sales number for each platform at a given year.

Second Visualization

In this second visualization, I chose to use a scatter plot to display the relationship between the year of release and global sales of video games, with each point representing a game. To provide more information about each game, I added tooltips that show the game's name, platform, year of release, genre, publisher, and global sales when the user hovers over a point. To make the visualization more interactive, I added a slider that allows the user to filter the games by year of release, and a dropdown menu that allows the user to filter the games by platform. The user can select a range of years using the slider, and choose one or more platforms using the dropdown menu. The scatter plot updates dynamically to show only the games that match the selected filters. I chose these interactive features because they allow the user to explore the dataset in a more flexible and interactive way. The slider and dropdown menu provide the user with the ability to filter the data by specific criteria, which helps them to focus on particular games of interest. By adding tooltips, the user can easily get more information about each game by hovering over a point. Overall, these interactive features provide a more engaging and informative way to explore the dataset.
